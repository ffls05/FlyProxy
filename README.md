## FlyProxy的三种套餐方案

他们家提供三种不同类型的代理服务，适合不同使用场景：

**动态住宅代理**

这是最常用的套餐类型，按流量计费。根据行业测评数据，价格大概是这样的：20GB套餐约2.5美元/GB，50GB套餐降到2.4美元/GB左右，如果购买大流量套餐，比如10TB级别的企业套餐，单价可以降到0.9美元/GB。

动态代理的特点是IP会自动轮换，每次请求都可能使用不同的IP地址。IP存活时间不超过24小时，适合需要大量不同IP的数据抓取场景。成功率能达到99%以上，响应速度大概在0.6秒左右。

**静态住宅代理**

如果你需要一个固定的IP地址保持较长时间，那静态住宅代理就是你要的。价格大约从5.5美元/IP/月起，具体价格会根据地区和数量有所浮动。

静态IP的好处是稳定，线路质量好，上网时间长。同时支持无限带宽和并发会话，对于需要保持账号登录状态、维护社交媒体账号这类场景特别合适。

**无限流量套餐**

这是按时间计费的套餐，在购买的时间段内可以无限制使用流量。官方提供1天、7天、30天等不同时长的选择，日套餐起价大约在68美元左右（这个价格在行业内属于中等偏上水平）。
